# Scene And Prefab Naming

This document defines required naming rules for scenes and prefabs used by the current project scope.

## Scene Naming
- Format: `<Domain>_<Name>`
- Examples:
  - `Core_Bootstrap.unity`
  - `Shop_BlacksmithGroundFloor.unity`
  - `Shop_BlacksmithUpperFloorHome.unity`
  - `World_TownSquare.unity`
  - `Interior_MinerShop.unity`
  - `Interior_WoodworkerShop.unity`
  - `Interior_TannerShop.unity`

## Prefab Naming
- Format: `<Category>_<Name>_PF`
- Examples:
  - `Player_Blacksmith_PF`
  - `NPC_Villager_PF`
  - `NPC_Thief_PF`
  - `NPC_AlchemistElder_PF`
  - `NPC_Beggar_PF`
  - `Companion_GuardDog_PF`
  - `Station_Furnace_PF`
  - `Station_Anvil_PF`
  - `Station_Grindstone_PF`
  - `Station_QuenchBucket_PF`
  - `Tool_Hammer_PF`
  - `Tool_Tongs_PF`

## Variant Naming
- Format: `<BasePrefabName>_V<Level>`
- Examples:
  - `Station_Furnace_PF_V1`
  - `Station_Furnace_PF_V2`
  - `Station_Furnace_PF_V3`
  - `Tool_Hammer_PF_V1`
  - `Tool_Hammer_PF_V2`
  - `Tool_Hammer_PF_V3`

## UI Prefab Naming
- Format: `UI_<Name>_PF`
- Examples:
  - `UI_DayClock_PF`
  - `UI_NpcSubtitle_PF`
  - `UI_Handbook_PF`
  - `UI_OrderPanel_PF`
